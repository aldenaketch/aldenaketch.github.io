Here’s my recent work.

These projects reflect real-world scenarios, penetration testing, and vulnerability assessment

Capture the Flag (CTF): Operation SMB Exploit

Analyze and exploit SMB vulnerabilities in Windows and Linux systems, simulating real-world cyberattacks like WannaCry or NotPetya. Gain hands-on experience with password cracking, brute-force SSH attacks, and SMB share enumeration to understand attacker techniques and recommend mitigations.

Penetration Testing Simulation

Conduct a full-cycle penetration test on a simulated network using Kali Linux, practicing reconnaissance, vulnerability scanning, exploitation, and reporting. This project emphasizes scoping, executing, and documenting findings to strengthen organizational security.

Vulnerability Assessment Lab

Perform comprehensive vulnerability scans on systems and applications to identify weaknesses, such as outdated software or misconfigurations. Develop skills in analyzing scan results and proposing actionable mitigation strategies.

Social Engineering Attack Simulation

Simulate phishing or pretexting attacks to test and improve defenses against social engineering tactics. This project teaches how to identify and counter human-based vulnerabilities in organizational security.

Web Application Security Testing

Assess web applications for common vulnerabilities like SQL injection or cross-site scripting (XSS) using ethical hacking tools. Practice identifying and securing web-based threats to protect digital assets.

Network Security Analysis and Firewall Configuration

Analyze network configurations and test firewall rules to prevent unauthorized access. This project involves hands-on labs to configure and optimize network security measures, ensuring robust defense against intrusions.

For each project, I have acquired skills such as Kali Linux proficiency, penetration testing, or vulnerability mitigation and their real-world relevance.
